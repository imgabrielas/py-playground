# py-playground

A collection of small Python projects — fun, experimental, or just things that didn't fit anywhere else. Code here tends to be short and simple by design.

## Projects

### [math_drawings](math_drawings/)
Simple Python scripts that generate drawings and visualisations using matplotlib plots. Mathematical patterns made visual.

### [street_crossing_game](street_crossing_game/)
A Frogger-inspired game built with Python's `turtle` module. Guide a turtle across a busy street while dodging oncoming cars. Difficulty increases with each successful crossing.
